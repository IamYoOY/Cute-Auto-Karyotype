# Cute Auto Karyotype✨

Version 1.0
First Public Release (2026)

Cute Auto Karyotype is a user-friendly software developed for chromosome detection, chromosome cropping, karyotype preparation, and image export from metaphase chromosome spreads.

The software was designed to simplify routine cytogenetic workflows by providing automatic chromosome detection, manual editing tools, chromosome numbering, crop refinement, and final karyotype board generation within a single application.

## Features

* Automatic chromosome detection
* Keep / Reject chromosome selection
* Manual chromosome box creation
* Chromosome numbering by size or position
* Crop editing with Brush and Lasso tools
* Chromosome rotation with custom angle input
* Undo / Redo support
* Final Karyotype Board generation
* PNG and PDF export
* Layer-based chromosome crop export
* User-friendly graphical interface

## Installation

### Step 1. Install Python

Download Python from:

https://www.python.org/downloads/

During installation, enable:

☑ Add Python to PATH

---

### Step 2. Install Required Packages

Double-click:

INSTALL_Cute_Karyotype.bat

The installer will automatically install all required Python packages.

---

### Step 3. Run the Program

Double-click:

RUN_Cute_Karyotype.bat

The program window should open automatically.

---

## Recommended Workflow

1. Click Detect to automatically identify chromosomes in the image.
2. Review detected chromosome boxes and Keep or Reject them as needed.
3. Use Add Box to manually add missing chromosome regions.
4. Click Delete Rejected to remove unwanted boxes.
5. Use Auto Number to assign chromosome numbers.
6. Edit chromosome crops using Brush or Lasso tools.
7. Rotate chromosomes if necessary.
8. Export results as PNG or PDF.
9. Use Create Final Board to generate the final karyotype layout.

## Troubleshooting

### Program does not start

Run:

INSTALL_Cute_Karyotype.bat

again and verify that Python is installed correctly.

### Export fails

Close any exported PNG or PDF files before exporting again.

### User interface appears incomplete

Set Windows display scaling between 100% and 150%.
